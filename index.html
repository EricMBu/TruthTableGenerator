Instructions

Operators:

  • ! - not
  • ^ - and
  • v - or
  • * - xor
  • = - if only if
  • > - implies

Type out these characters, or use the buttons underneath the expression box

Use parentheses and variables p, q, r, and s in expression

Note: the 'implies' operator currently has the lowest precedence due to a bug, please use parantheses to specify parts of your expression when using the '>' operator
